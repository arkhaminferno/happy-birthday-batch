# CelebrateVibes After Effects template

Shipped with the batch repo so Mac and Windows clones can render without extra setup.

- **Project:** `Happy birthday.aep`
- **Assets:** `assets/` (video, logo, effects)
- **Edit comp:** `Happy Birthday` (name text layer `Rajesh`)
- **Render comp:** `MAIN 2Min+` (1080p YouTube export)

Used by `python -m batch_birthday ae-batch`. Large media files are stored with Git LFS.
